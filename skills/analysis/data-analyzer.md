# Data Analyzer Skill

## Description
A skill for analyzing datasets, identifying patterns, and generating insights from structured data.

## Purpose
Help users understand their data through statistical analysis, pattern recognition, and actionable insights.

## Usage

### Prompt Template
```
Analyze the following data:

Data Type: [e.g., CSV, JSON, time series]
Context: [What the data represents]
Questions: [Specific questions or goals]

[Insert data or data description]

Please provide:
- Summary statistics
- Key patterns or trends
- Anomalies or outliers
- Insights and recommendations
```

## Example

**Input:**
```
Analyze monthly sales data:
Jan: $45K, Feb: $52K, Mar: $48K, Apr: $65K, May: $78K, Jun: $92K

Question: What trends do you see and what should we focus on?
```

**Expected Output:**
- Growth rate calculation
- Trend identification (positive growth)
- Seasonality observations
- Forecasting suggestions
- Action items based on trends

## Best Practices
- Provide context about the data source
- Specify what questions you want answered
- Include relevant metadata (time periods, units, etc.)
- Mention any known data quality issues
