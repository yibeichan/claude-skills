# Claude Skills Collection

A curated collection of custom skills, prompts, and tools for Claude AI assistant.

## Overview

This repository contains a collection of reusable skills that enhance Claude's capabilities. Each skill is designed to help Claude perform specific tasks more effectively, whether it's code generation, data analysis, creative writing, or problem-solving.

## Structure

```
claude-skills/
├── skills/              # Individual skill definitions
│   ├── coding/         # Programming and development skills
│   ├── writing/        # Content creation and writing skills
│   ├── analysis/       # Data analysis and research skills
│   └── productivity/   # Task management and productivity skills
├── examples/           # Example usage of skills
└── docs/              # Additional documentation
```

## Available Skills

See the **[Skills Catalog](CATALOG.md)** for a complete list of available skills organized by category.

Current skill categories:
- **Coding** (2 skills) - Code review, test generation, and development assistance
- **Writing** (1 skill) - Technical documentation and content creation
- **Analysis** (1 skill) - Data analysis and insights
- **Productivity** (1 skill) - Meeting summaries and task management

Each skill contains:
- A description of what it does
- Usage instructions with prompt templates
- Example prompts and expected outcomes
- Best practices for effective use

## Quick Start

1. Browse the **[Skills Catalog](CATALOG.md)** to find a skill that matches your needs
2. Open the skill's documentation to understand its purpose
3. Copy the prompt template from the skill file
4. Customize the template with your specific context
5. Use it with Claude in your preferred interface (Claude.ai, API, etc.)
